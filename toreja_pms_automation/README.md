# Toreja Medical Clinic PMS — Selenium Python Automation

Automated test suite for **http://torejamedicalclinic.wuaze.com** using
**Selenium WebDriver + pytest**, structured with the **Page Object Model (POM)**.

---

## Quick Start

### 1. Install dependencies

```bash
pip install -r requirements.txt
```

### 2. Configure environment

Edit `.env` to set your browser and credentials:

```
BASE_URL=http://torejamedicalclinic.wuaze.com
ADMIN_USERNAME=admin
ADMIN_PASSWORD=admin123
BROWSER=chrome
HEADLESS=false
```

### 3. Run all tests

```bash
pytest
```

### 4. Run a specific module

```bash
pytest tests/test_login.py
pytest tests/test_patients.py
```

### 5. Run with HTML report

```bash
pytest --html=reports/report.html --self-contained-html
```

### 6. Run headless (for CI)

Set `HEADLESS=true` in `.env`, or:

```bash
HEADLESS=true pytest
```

### 7. Run by test ID pattern

```bash
pytest -k "login"
pytest -k "TC-PAT"
```

## Key Design Decisions

| Concept | Implementation |
|---|---|
| **Page Object Model** | Every page is a class in `pages/`. Tests never call `find_element` directly. |
| **Central Locators** | All `By.ID`, `By.NAME`, `By.XPATH` etc. live in `locators/locators.py` only. |
| **Shared Fixtures** | `conftest.py` provides `driver`, `auth_driver`, and a ready-opened page fixture per module. |
| **Test Data** | All strings/values are in `test_data/test_data.py` — no magic strings in tests. |
| **Auto-screenshot** | `conftest.py` hooks into pytest to save a screenshot on every test failure. |
| **Select2 support** | `helpers.py` has `select2_choose()` and pages handle JS dropdowns via click-then-type. |
| **Date pickers** | Bypassed using `helpers.set_date_via_js()` to set values directly via JavaScript. |

---

## Notes on App-Specific Quirks

- **Select2 dropdowns** (Patient, Medicine selects): Click the Select2 trigger span → type in the search box → click the matching option.
- **Custom date pickers**: Use JS executor to set values — `driver.execute_script("arguments[0].value = ...", el)`.
- **DataTables**: Rows load dynamically. Always use `WebDriverWait` before asserting row counts.
- **Shared hosting**: The app can be slow. Adjust `EXPLICIT_WAIT` in `.env` if you see timeout failures.
